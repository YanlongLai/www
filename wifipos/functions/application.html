WifiPOS

Situational application

  1. Can support wireless-order in guest room
  2. Simultaneous orderings by multi-waiters
  3. Learning quickly and using intuitively
  4. Using KIOSK system at front door
  5. Have additional combination with coin machine
  6. Self-ordering via scanning QR Code on guest table
  7. After waiter opens table, guest can directly confirm and send out order
  8. Each table has one tablet for ordering
  9. Remote ordering available in wifi area
  10. Support different languages in different sections
